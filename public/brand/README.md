# Football 2027 Brand Assets

This directory contains the **shipping vector sources** for the Football 2027 identity.

## Source of truth

- **Visual specification:** Football 2027 Brand Kit & Visual Language in Google Drive.
- **Shipping assets and token implementation:** this GitHub repository.
- **Exploration and editable marketing layouts:** Canva/Figma.

## Included files

| File | Intended use |
|---|---|
| `football-2027-mark.svg` | Primary two-colour icon on dark or transparent surfaces. |
| `football-2027-mark-white.svg` | Small/noisy surfaces, scorebug watermark and one-colour use. |
| `football-2027-lockup-horizontal.svg` | Splash, main menu, project pages and wide marketing placements. |
| `football-2027-app-icon.svg` | Master vector for app icon and social-avatar raster exports. |
| `football-2027-splash-background.svg` | Scalable decorative background for loading and menu screens. |

## Working concept: Forward Pitch

The mark combines:

1. an angular **F**;
2. a forward-rising diagonal representing progress through the football pyramid;
3. a centre-circle detail.

It intentionally avoids club-crest framing, football-ball clichés, licensed competition shapes and broadcaster imitation.

## Core colours

- Night Navy `#07101C`
- Deep Panel `#0E1B2B`
- Signal Cyan `#14D1E6`
- Progress Green `#33D677`
- Primary Text `#F2F7FF`

## Usage rules

- Preserve clear space around the mark equal to the centre-circle diameter.
- Prefer the two-colour mark on Night Navy or transparent dark surfaces.
- Use the white mark below roughly 32 px or over visually busy imagery.
- Do not stretch, bevel, outline, rotate or place the mark inside a generic football crest.
- Do not recolour the primary mark to resemble a real club when it appears beside team identity.
- Cyan means interaction/system focus; green means success/progression. Do not apply both to every control.

## Raster export backlog

The SVG files are authoritative. Before a public release, export and verify:

- `football-2027-mark-transparent-1024.png`
- `football-2027-app-icon-512.png`
- `football-2027-favicon-32.png`
- optional 16 px favicon and 256 px social avatar

Check legibility at 16, 32, 64 and 256 px, on dark and temporary light backgrounds.
