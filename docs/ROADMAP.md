# Roadmap

## Phase 1: Foundation (Completed)
- Audit existing code
- Establish fixed timestep loop
- Setup proper input buffering and controller abstraction
- Document architecture
- Add basic Vitest coverage

## Phase 2: Expressive Control (Next)
- Advanced First Touch logic
- Shielding interactions
- Pass and shot context targeting
- Full Input intent parsing

## Phase 3: Defensive & AI Mechanics
- Improve Goalkeeper logic (diving, parrying)
- Sliding and standing tackle windows
- Jockeying
- Lightweight `TacticalPolicy` integration for opponents

## Phase 4: Multiplayer & Scale
- Scale 1v1 up to 11v11
- Spatial hashing for collisions
- Tactical shape and off-ball runs

## Phase 5: Career Architecture
- Zod schemas for Clubs, Players, Competitions
- SQLite/IndexedDB persistence
- Gemini AI scouting reports and event generation
